# Registry Member Detail Drawer (operations)

<!-- help:component-source -->
_Linked component: `client/src/components/operations/RegistryMemberDetailDrawer.tsx`._
<!-- /help:component-source -->

## What this is

The Registry Member Detail Drawer provides a focused view of an individual wedding party member's status. It is accessible from the Registry Priority Feed and the main Registry Dashboard.

## How to use it

1. Review measurements and retail sizing.
2. View payment status and ledger summary.
3. Check upcoming appointments.
4. Access the full wedding party file for deeper management.

## Tips

- Use this drawer for quick status checks without leaving the dashboard.
- Direct measurement edits captured here sync to the CRM vault and the wedding registry.
