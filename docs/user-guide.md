# User Guide for Main Shell Component

## Overview

The main shell component is the central hub of the application, responsible for managing UI state and rendering content based on user interactions and application state. It supports various modes such as POS (Point of Sale), Insights (analytics and reporting tools), and Wedding (wedding-related functionalities). The component includes features like sidebar navigation, global search drawer, deep link handling, modals for closing the register, drawers for help center and bug reports, permissions and access control, theme mode, error handling, and navigation and routing.

## Navigation

### Sidebar Navigation

1. **Open the Sidebar**: Click on the hamburger icon in the top-left corner of the screen.
2. **Select a Section**: Choose from the available sections such as POS, Insights, Wedding, etc., to navigate between different parts of the application.

### Global Search Drawer

1. **Open the Search Drawer**: Click on the search icon in the top-right corner of the screen.
2. **Enter Search Query**: Type your query to search for customers, products, and wedding party customers.
3. **Select a Result**: Click on the desired result to view more details.

### Deep Link Handling

1. **Access via URL**: Use deep links to directly access specific features like alterations, procurement, inventory product hub, QBO sync logs, etc.
2. **Example**: `https://app.example.com/alterations` will take you directly to the alterations section.

## Modes

### POS Mode

1. **Switch to POS Mode**: Click on the "POS" tab in the sidebar.
2. **Use POS Features**: Access and use the Point of Sale interface for transactions and sales.

### Insights Mode

1. **Switch to Insights Mode**: Click on the "Insights" tab in the sidebar.
2. **View Analytics**: Explore analytics and reporting tools to gain insights into your business operations.

### Wedding Mode

1. **Switch to Wedding Mode**: Click on the "Wedding" tab in the sidebar.
2. **Manage Wedding**: Use wedding-related functionalities to manage events and customers.

## Permissions and Access Control

- **Access Control**: The application ensures that users can only access features they are authorized to use based on their permissions.

## Theme Mode

- **Switch Themes**: Click on the theme icon in the top-right corner of the screen to switch between light and dark themes.

## Error Handling and Loading States

- **Loading States**: The application displays loading indicators while data is being fetched.
- **Error Handling**: If an error occurs, an error message will be displayed, and you can try again or contact support for assistance.

## Modals and Drawers

### Modals

1. **Close Register Modal**: Click on the "Close Register" button to open the modal.
2. **Confirm Action**: Follow the prompts to confirm closing the register.

### Drawers

1. **Help Center Drawer**: Click on the help icon in the top-right corner to open the help center drawer.
2. **Bug Report Drawer**: Click on the bug report icon in the top-right corner to open the bug report drawer.

## Conclusion

By following these instructions, you can effectively navigate and utilize the main shell component of the application to manage your business operations efficiently.
