---

## End-of-day questions

1. Did any customer leave without a clear answer?
2. Did any payment, refund, or balance require manual review?
3. Did any workflow stay unfinished at close?
4. Did any staff member avoid a feature because it felt risky or confusing?
5. Did any manager override happen more than once for the same reason?
6. Did any register close step require explanation or rework?
7. Does tomorrow need a temporary pilot rule?
