# Riverside OS v0.60.2

Riverside OS v0.60.2 is a targeted release focused on modernizing the installation, updates, and maintenance interface.

## Highlights

- **Modernized Deployment Manager**: Rebuilds the old PowerShell/WinForms deployment interface into an interactive React + Tauri desktop wizard.
- **Port Isolation**: Binds the manager's development server to port `5179` to prevent conflicts with the main POS port (`5173`).
- **Server Control Panel**: Integrates quick buttons to start, restart, check, and open server logs natively in Windows File Explorer.
- **Database & Migrations Workspace**: Adds direct controls to execute pending schema migrations, seed database tables, and trigger a factory reset.
- **Utility Actions**: Bundles one-click execution for ROSIE AI stack updates, Counterpoint bridge token mapping, signing credential repair, and administrator recovery.
- **Parity Alignment**: Automatically synchronizes configuration schema and bumps all metadata values to `v0.60.2`.

## Deployment

Install or update using only the `v0.60.2` signed Windows updater/deployment assets. Confirm **Settings -> Updates** shows Riverside `0.60.2` across the Windows app, server API, and PWA/web files before live sales.
