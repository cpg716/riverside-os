# Riverside OS v0.60.0

Riverside OS v0.60.0 is the production release for the Riverside Men's Shop rollout. It aligns the server, PWA/web app, Windows desktop shell, updater metadata, Help Center, ROSIE grounding, and deployment package under one release identity.

## What changed

- Hardened Register close and operational readiness for daily store use.
- Improved reporting/QBO support surfaces for accounting review.
- Continued Podium communication and provider sync hardening.
- Recertified Help Center and staff guidance against the current workflows.
- Aligned Windows updater and deployment-package defaults to `0.60.0`.
- Labeled existing bug-report JSON exports as AI diagnostic packages for Codex-ready support handoff.

## Validation

See `docs/releases/v0.60.0-certification.md` for the release evidence, including local gates, CI status, release workflow results, artifact verification, and remaining human/environment signoffs.

## Operator notes

- Install only the `v0.60.0` Windows updater/deployment assets from this release.
- Confirm **Settings -> Updates** shows the same Riverside version for Windows app, PWA/web files, and server API.
- Keep the previous installer and a fresh database backup available until first-week observation is complete.
