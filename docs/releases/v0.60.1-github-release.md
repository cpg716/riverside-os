# Riverside OS v0.60.1

Riverside OS v0.60.1 is a focused deployment hotfix for the production store rollout.

## Highlights

- Adds known-host choices to sign-in **API Host Settings** while preserving manual IP/URL entry.
- Keeps the Backoffice / Server desktop app on the direct local server path.
- Adds PWA/browser receipt-printer readiness checks through the Riverside server-to-printer TCP path.
- Keeps receipt and tag printing on direct hardware paths while reports continue through browser/Windows print.
- Aligns package, server, Tauri, and deployment metadata to `0.60.1`.

## Deployment

Install or update using only the `v0.60.1` signed Windows updater/deployment assets. Confirm **Settings -> Updates** shows Riverside `0.60.1` across the Windows app, server API, and PWA/web files before live sales.
