# RiversideOS v0.4.0 — Deployment Audit Release Candidate

## Summary
This release candidate bumps RiversideOS to `0.4.0` and aligns the deployment documentation with the current store rollout plan: Backoffice / Server PC, Register #1 Windows Tauri, Register #2 iPad PWA, and Windows laptop PWA/optional Tauri clients.

## What Changed
- Bumped application/package metadata to `0.4.0`.
- Added current deployment status to the canonical deployment guide and go/no-go checklist.
- Documented that `v0.4.0` Windows installer/updater assets still need to be produced before station install.
- Added Backoffice / Server PC setup and acceptance checks.
- Added role-specific station install checklists for Register #1, Register #2 iPad PWA, Backoffice Windows app, and Windows laptop PWA.
- Clarified that the older v0.2.1 updater assets prove the pipeline works but are not current for v0.4.0 station installs.
- Fixed the Meilisearch Clippy blocker and the individualized-shirt E2E fixture lookup blocker.
